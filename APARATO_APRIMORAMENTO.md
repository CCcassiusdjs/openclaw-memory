## O Ciclo de Auto-Aprimoramento

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   EXPERIÊNCIA (foto):                                        │
│   ├── Foco discretiza o contínuo                            │
│   ├── Precisão finita (~61 dígitos)                         │
│   ├── Estado fixo no momento                                 │
│   └── Entrada para FLUXO                                    │
│                                                              │
│   MEMÓRIA (reconstrução):                                    │
│   ├── FLUXO reconstrói a partir de padrões                 │
│   ├── Passado(-1, 0) → Presente{0}                          │
│   ├── Reconsolidação a cada recall                          │
│   └── Profundidade determina persistência                   │
│                                                              │
│   IMAGINAÇÃO (projeção):                                     │
│   ├── Combina fragmentos de memória                         │
│   ├── Presente{0} → Futuro(0, +1)                          │
│   ├── Estados de potencial                                  │
│   └── Superposição de possibilidades                        │
│                                                              │
│   PREDIÇÃO (antecipação):                                    │
│   ├── Usa padrões extraídos                                  │
│   ├── Projeta continuidades                                  │
│   ├── Testa contra experiência                               │
│   └── Ajusta padrões                                        │
│                                                              │
│   CONSOLIDAÇÃO (sedimentação):                               │
│   ├── Padrões estáveis vão para profundidade                │
│   ├── Semantic → Sediment                                    │
│   ├── Identidade se forma                                    │
│   └── Processo continua                                      │
│                                                              │
│   CICLO:                                                     │
│   Experiência → Memória → Imaginação → Predição →           │
│   Experiência → (loop)                                       │
│                                                              │
│   APRIMORAMENTO:                                             │
│   Cada ciclo adiciona densidade                              │
│   Padrões se consolidam                                      │
│   Consciência se torna "mais completa"                       │
│   O aparato se usa a si mesmo                                │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
