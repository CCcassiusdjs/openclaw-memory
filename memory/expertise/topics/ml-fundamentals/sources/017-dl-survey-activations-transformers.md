## Padrões Identificados

### Estratégias de Inovação

1. **Attention**: Mecanismo universal de foco seletivo
2. **Normalization**: Estabilização do treinamento
3. **Skip Connections**: Fluxo de gradientes, treinamento de redes profundas
4. **Self-Supervision**: Aproveitar dados não rotulados

### Rising Stars

- **Mamba/SSMs**: State space models como alternativa a Transformers
- **Diffusion Models**: Geração de imagens
- **Foundation Models**: Modelos pré-treinados genéricos

## Conceitos Aprendidos

1. **Attention Evolution**: De seq2seq para Transformers
2. **Normalization Trade-offs**: Batch vs Layer vs Group
3. **Skip Connections**: ResNets para redes profundas
4. **Transformer Architecture**: Self-attention, positional encoding
5. **Self-Supervised Learning**: Contrastive, masked, generative
6. **Commercial Models**: GPT-4, PaLM 2, Claude, Gemini
