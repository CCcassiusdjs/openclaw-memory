## Visão Geral

MLflow é uma plataforma open-source para gerenciar o ciclo de vida completo de machine learning, incluindo experimentação, reprodutibilidade, deployment e monitoramento. O componente MLflow Tracking é o núcleo para registro de parâmetros, métricas, artefatos e modelos.

## Conceitos-Chave a Estudar

1. **Tracking API**: `mlflow.start_run()`, `mlflow.log_param()`, `mlflow.log_metric()`, `mlflow.log_artifact()`
2. **Experiments e Runs**: Organização de experimentos, runs aninhados (parent/child)
3. **Autologging**: Integração automática com sklearn, pytorch, tensorflow, xgboost
4. **Model Registry**: Versionamento de modelos, stages (staging, production, archived)
5. **Artifacts**: Armazenamento de modelos, gráficos, datasets
6. **Backend Stores**: SQLite, PostgreSQL, MySQL para metadata
7. **Artifact Stores**: Local filesystem, S3, GCS, Azure Blob
8. **Tracking Server**: Deploy standalone, Databricks, cloud
9. **Projects**: Empacotamento de código ML reprodutível
10. **Model Packaging**: MLflow Models, flavors, deployment
11. **Tags e Notes**: Metadados customizados para runs
12. **Dataset Tracking**: Dataset lineage e versioning
13. **GenAI Integration**: Tracing, prompts, LLM evaluation
