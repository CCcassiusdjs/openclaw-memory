### Why Redundancy Failed Under Radiation

**Root Cause:** Shared I2C bus = Single Point of Failure

| Component | Bus | Sensors Affected |
|-----------|-----|------------------|
| I2C | Single | BMP280×3 (BARO), AK09916×3 (MAG) |
| SPI | Single | ICM20602×3 (IMU) |
| GPS | UART | Single GPS |

**Failure Mode:**
1. SEU flips bit on I2C bus
2. ALL BARO/MAG sensors receive corrupted data
3. All cores see: `hgtTestRatio → ∞`, `magTestRatio → ∞`
4. All cores: `errorScore = MAX(∞, ∞, ∞) = ∞`
5. `coreRelativeErrors[i] = inf - inf = NaN`
6. Core selection fails → indeterminate state

### Recommendations for True Radiation Tolerance

1. Physically separate I2C/SPI buses per sensor
2. Independent sensor data per EKF core
3. Bus corruption detection mechanism
4. Fallback to healthy sensor subset
5. Watchdog for core selection deadlock

### Key Insight

GPS failures (83%) exceed IMU failures (78%), suggesting:
1. GPS UART is more susceptible to SEU than SPI
2. OR GPS processing software has less error tolerance
