# Creativity and Machine Learning: A Survey (arXiv:2104.02726)

**Fonte:** https://arxiv.org/abs/2104.02726
**Autores:** Giorgio Franceschelli
**Ano:** 2024
**Publicado:** ACM Computing Surveys, 56(11)
**Status:** Completed

---

## Resumo

Survey sobre criatividade computacional e machine learning, cobrindo teorias, técnicas e métodos de avaliação.

---

## Estrutura

### 1. Teorias de Criatividade Computacional

#### Definições de Criatividade
- **Novelty + Value**: Novo e útil
- **Transformation**: Transformar espaço de problemas
- **Exploration vs Exploitation**: Busca criativa

#### Tipos de Criatividade
- **P-creativity**: Pessoal/psicológica
- **H-creativity**: Histórica
- **Combinational**: Combinação de ideias
- **Exploratory**: Exploração de espaço conceitual
- **Transformational**: Transformação de regras

### 2. Técnicas de ML para Criatividade

#### Generative Deep Learning
- **VAE**: Variational Autoencoders
- **GAN**: Generative Adversarial Networks
- **Diffusion Models**: Stable Diffusion, DALL-E
- **Flow-based models**: Normalizing flows

#### Aplicações Criativas
- **Text**: Language models, story generation
- **Image**: Art generation, style transfer
- **Music**: Composition, style transfer
- **Video**: Generation, editing
- **Code**: Program synthesis

### 3. Métodos de Avaliação

#### Avaliação Automática
- **Metrics**: Inception Score, FID
- **Diversity**: Coverage, uniqueness
- **Quality**: Fidelity, coherence

#### Avaliação Humana
- **Turing Test**: Human vs Machine
- **Preference**: Human judgment
- **Creativity Tests**: Torrance Tests

### 4. Desafios

#### Técnicos
- **Evaluation**: Como avaliar criatividade?
- **Control**: Controle sobre output
- **Novelty vs Quality**: Trade-off

#### Filosóficos
- **Authorship**: Quem é o autor?
- **Originality**: O que é original?
- **Value**: Como definir valor?

### 5. Oportunidades Futuras

1. **Human-AI Collaboration**: Co-criação
2. **Interactive Systems**: Feedback em tempo real
3. **Multimodal Generation**: Text + Image + Audio
4. **Personalization**: Criatividade personalizada
5. **Ethical Considerations**: Bias, ownership

---

## Conceitos Aprendidos

1. **Creativity Types**: Combinational, exploratory, transformational
2. **Generative Models**: VAE, GAN, Diffusion, Flow
3. **Evaluation Methods**: Automatic metrics, human evaluation
4. **Creative Applications**: Text, image, music, video, code
5. **Challenges**: Evaluation, control, novelty vs quality
6. **Future Directions**: Human-AI collaboration, multimodal, ethics
