## Sobre o Livro

Este é um livro de nível graduate focado na **análise e teoria de algoritmos de machine learning**. Serve como livro-texto para estudantes de pós-graduação e referência para pesquisadores.

## Conceitos-Chave

### Complexidade de Rademacher
- Medida de capacidade de uma classe de funções
- Alternativa à dimensão VC
- Bounds mais apertados para generalização

### Dimensão VC (Vapnik-Chervonenkis)
- Capacidade de "shattering" de uma classe de hipóteses
- Fundamental para PAC learning
- Bounds de generalização baseados em VC

### Margem e Boosting
- Margin bounds para generalização
- AdaBoost como maximização de margem
- Conexão entre boosting e margin

### Online Learning
- Regret minimization
 - Online-to-batch conversion
- Perceptron e variantes
