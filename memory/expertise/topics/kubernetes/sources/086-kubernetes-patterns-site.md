## Padrões Kubernetes

O livro cobre padrões reutilizáveis para projetar aplicações cloud-native:

### Categorias de Padrões
1. **Foundational Patterns**
   - Health Check
   - Predictable Behavior
   - Automated Placement

2. **Structural Patterns**
   - Init Container
   - Sidecar
   - Ambassador
   - Adapter

3. **Behavioral Patterns**
   - Batch Job
   - Scheduled Job
   - Daemon Service
   - Service Discovery

4. **Configuration Patterns**
   - ConfigMap
   - Secret
   - Immutable Configuration

5. **Security Patterns**
   - Pod Security
   - Network Security
   - RBAC

6. **State Management Patterns**
   - Persistent Storage
   - StatefulSet
   - Volume Claims

## Conceitos-Chave Extraídos

| Padrão | Descrição |
|--------|-----------|
| Sidecar | Container auxiliar no mesmo Pod |
| Ambassador | Proxy para serviços externos |
| Adapter | Normaliza interface do container |
| Init Container | Executa antes do container principal |
