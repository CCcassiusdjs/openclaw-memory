## 🎯 Sobre Neptune

**Neptune.ai** foi uma plataforma de **experiment tracking** para ML:
- Focused on hands-on, iterative model development
- Track experiments, monitor training, analyze complex behavior
- **Acquired by OpenAI** (March 2026)

## 📋 Features Principais

| Feature | Descrição |
|---------|-----------|
| **Experiment Tracking** | Log params, metrics, artifacts |
| **Training Monitoring** | Real-time training visualization |
| **Run Comparison** | Compare thousands of runs |
| **Layer Analysis** | Analyze metrics across layers |
| **Issue Detection** | Surface problems automatically |

## 🔄 Integração com OpenAI

OpenAI acquired Neptune para:
- Strengthen tools for frontier research
- Integrate deep into training stack
- Expand visibility into how models learn
- Analyze complex training workflows
