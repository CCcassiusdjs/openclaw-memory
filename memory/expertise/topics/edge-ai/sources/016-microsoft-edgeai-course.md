## SLMs (Small Language Models)

### Modelos Cobertos
- **Phi Family** (Microsoft)
- **Qwen Family**
- **Gemma** (Google)
- **BitNET**
- **μModel**
- **Phi-Silica**

### Características de SLMs
- Menor footprint de memória
- Menor demanda computacional
- Inicialização mais rápida
- Otimizados para CPU e edge GPU

---

## Por Que Edge AI?

### Vantagens
1. **Privacidade**: Dados não saem do dispositivo
2. **Latência**: Sem latência de rede
3. **Custo**: Menor uso de cloud/bandwidth
4. **Resiliência**: Funciona offline
5. **Compliance**: Atende requisitos de soberania de dados

---

## Otimização de Modelos

### Técnicas Cobertas
- **Quantização**: Redução de precisão (Float32 → Int8)
- **Llama.cpp**: Framework de inferência otimizado
- **Microsoft Olive**: Ferramenta de otimização
- **OpenVINO**: Intel - otimização para hardware Intel
- **Apple MLX**: Framework Apple para M-series

### Métricas de Melhoria
- 85% speed boost
- 75% size reduction
