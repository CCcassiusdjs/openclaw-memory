## Problema Identificado

**Hard-switch `lan`** agrupa fisicamente lan1+lan2+lan3 no chip do FortiGate.

Para usar `lan1` como porta de manutenção independente:
1. Criar lan1 como interface física (requer IP temporário diferente)
2. Configurar políticas firewall lan1 ↔ VLANs
3. Configurar DHCP server lan1
4. **Remover hard-switch `lan`** (derruba SSH atual)
5. Migrar lan1 para IP final 192.168.1.99/24

**Risco:** Perda de acesso SSH durante transição. Requer console serial ou GUI web.
