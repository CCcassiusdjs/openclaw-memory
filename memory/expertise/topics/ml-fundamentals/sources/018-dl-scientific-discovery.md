**Status:** Completed

## Resumo

Survey abrangente sobre aplicações de deep learning em descoberta científica, cobrindo visual, sequencial e dados estruturados em grafos.

---

## Estrutura

### 1. Modelos de Deep Learning

#### Visual Data
- **CNNs**: Arquiteturas para imagens
- **Vision Transformers**: Self-attention em imagens
- **Applications**: Medical imaging, microscopy, astronomy

#### Sequential Data
- **RNNs/LSTMs**: Sequências temporais
- **Transformers**: Attention para sequências
- **Applications**: Genomics, physics simulations

#### Graph-Structured Data
- **GNNs**: Graph Neural Networks
- **Message Passing**: Propagação de informação
- **Applications**: Molecular structures, social networks

### 2. Training Methods

#### Supervised Learning
- Standard training with labels
- Transfer learning
- Fine-tuning

#### Semi-Supervised Learning
- Pseudo-labeling
- Consistency regularization
- Self-training

#### Self-Supervised Learning
- Contrastive learning
- Masked modeling
- Pre-training objectives

### 3. Challenges em Ciência

#### Data Efficiency
- Few-shot learning
- Transfer learning
- Data augmentation
- Active learning

#### Interpretability
- Attention visualization
- Feature attribution
- Saliency maps
- Concept activation vectors

### 4. Design Process

#### Pipeline Completo
1. **Data preparation**: Collection, cleaning, preprocessing
2. **Model selection**: Architecture choice
3. **Training**: Hyperparameter tuning
4. **Evaluation**: Metrics, validation
5. **Deployment**: Production considerations
